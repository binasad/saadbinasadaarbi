Hosted on GitHub Pages: [https://binasad.github.io/personal-website](https://binasad.github.io/personal-website)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

Then visit `http://localhost:8000`
